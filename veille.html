Veille Technologique

Mon Sujet d'Étude

« Les impacts de l'Intelligence Artificielle sur le Web et la Cybersécurité ».

L'IA générative transforme le développement web (outils, automatisation) tout en créant de nouveaux risques majeurs (Deepfakes, Phishing ciblé, vol de propriété intellectuelle) que je dois surveiller activement pour mon futur métier de développeur.

Ma Méthodologie

1. Collecte (Feedly)

Agrégation automatisée de flux RSS de sources de référence (TechCrunch, The Verge, L'Usine Digitale, Le Monde Informatique).

2. Analyse & Stockage

Utilisation de l'extension Notion Web Clipper pour capturer l'info et l'analyser directement dans mon tableau de bord.

3. Synthèse Dynamique

Publication d'une version publique de ma base de données Notion, mise à jour automatiquement pour le jury.

Base de données complète

Pour le BTS, j'ai centralisé l'intégralité de ma veille, de mes analyses et de mes projections techniques sur un tableau de bord Notion interactif et mis à jour en continu.

Siphonnage d'IA : Anthropic accuse des laboratoires

Source : TechCrunch (via Feedly)

Anthropic accuse trois entreprises d'IA d'avoir créé des milliers de faux comptes pour "siphonner" les capacités de codage et de raisonnement de son modèle Claude via une technique de "distillation".

Mon analyse (celle que tu aimais) : Je trouve cela inquiétant pour la propriété intellectuelle, car cela montre que même les API sécurisées peuvent être détournées pour cloner des technologies propriétaires. Cela pose un vrai problème de cybersécurité technique.
Projection Technique (SLAM) : Pour mes développements, cela confirme la nécessité d'implémenter des limites de requêtes (Rate Limiting) et de la détection d'bots pour protéger l'accès à mes API.

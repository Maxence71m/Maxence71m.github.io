JobPilot — Plateforme de recrutement

Laravel PHP MySQL API REST

💼 Contexte & Objectifs

JobPilot est une plateforme web de gestion d'offres d'emploi développée dans le cadre de ma formation en BTS SIO option SLAM.

L'objectif du projet était de créer une application permettant aux entreprises de publier des offres d'emploi et aux candidats de postuler directement en ligne.

Le projet inclut également un système d'administration permettant de gérer les utilisateurs et leurs rôles.

⚙️ Fonctionnalités développées

  • Création et gestion des offres d'emploi (CRUD complet).
  • Gestion des candidats et des entreprises.
  • Système d'authentification sécurisé avec Laravel.
  • Gestion des rôles : admin, entreprise et candidat.
  • Possibilité pour les candidats de postuler à une offre.
  • Interface d'administration pour gérer les utilisateurs.
  • Création d'une API REST sécurisée pour accéder aux données.
  • Mise en place de logs d'audit pour suivre les actions importantes.

🚀 Difficultés et Solutions

Le problème : La gestion des rôles utilisateurs et des permissions nécessitait de sécuriser correctement les routes et les actions disponibles selon le type d'utilisateur.

Ma solution : J'ai implémenté un système de rôles dans la base de données et utilisé des middlewares Laravel pour limiter l'accès aux fonctionnalités sensibles (administration, modification d'offres, etc.).

J'ai également mis en place des Observers et des logs d'audit afin de tracer les actions importantes comme la création, la modification ou la suppression d'offres.

CRM Métier sur mesure

PHP / Laravel JS (Temps réel) Railway

🏢 Contexte & Enjeux

Ce projet a été réalisé pour l'entreprise Art & Sièges. L'entreprise utilisait des brouillons papier et des fichiers Excel, ce qui provoquait des erreurs de calcul.

Mon objectif : Développer un CRM métier sur mesure pour centraliser les clients, automatiser les calculs de prix complexes et suivre l'état des devis (En attente, Validé, Payé).

⚙️ Mes missions réalisées

  • Analyse : Étude des besoins de l'artisan pour traduire ses calculs papier en code.
  • Base de données : Création du schéma SQL pour lier les Clients et les Devis.
  • Backend : Développement de la logique avec Laravel.
  • Dynamisme : Création d'un calculateur en JavaScript pour afficher les prix en direct.
  • Documents : Mise en place de la génération automatique de devis en PDF.
  • Déploiement : Mise en ligne de l'application sur la plateforme Railway.

🚀 Difficultés et Solutions

1. Le rendu du PDF : La mise en page sautait souvent lors de la création du document A4.
Solution : J'ai créé un fichier de style (CSS) spécifique pour l'impression afin de bloquer le format.

2. Le calcul automatique : Il fallait que le total change dès qu'on tape un chiffre sans recharger la page.
Solution : Utilisation de JavaScript pour mettre à jour l'affichage instantanément.

3. La mise en ligne : Configurer la base de données à distance sur Railway était nouveau pour moi.
Solution : Paramétrage des "Variables d'environnement" pour sécuriser et réussir la connexion Cloud.
